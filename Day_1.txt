 Day 1: JavaScript Basics (History Style Notes)

---

1. What is JavaScript?

JavaScript is a programming language used to make websites interactive.

👉 Simple meaning:
HTML gives structure to a website
CSS gives design and style
JavaScript gives logic and interactivity

💡 What JavaScript can do:
Make buttons clickable
Validate forms
Create animations
Build games and apps
Make websites dynamic


## 🧠 2. Why JavaScript was created?

### 📜 The Problem (Before JavaScript)

In the early days of the internet:

* Websites were only **static pages**
* Users could only read content
* No interaction (no clicks, no forms working dynamically)

👉 Example:
If you clicked a button → nothing special happened

So the web was **boring and non-interactive**

---

## 👨‍💻 3. Who developed JavaScript?

JavaScript was created by:

👉 **Brendan Eich - 1995

* He worked at a company called **Netscape**
* In **1995**, he developed JavaScript
* He created it in just **10 days**

---

## ⚡ 4. Why it was created so fast?

Netscape wanted:

* A **lightweight programming language**
* To make web pages interactive
* Something easy for beginners

👉 So Brendan Eich built JavaScript very quickly to meet demand

---

## 📈 4. How JavaScript evolved?

### 📅 Timeline idea:

* 1995 → JavaScript created (for browser interactivity)
* Later → Became standard web language
* Now → Used for:

  * Web apps  
  * Mobile apps
  * Backend (Node.js)
  * Games & AI tools

---

## 🧠 5. Simple Understanding

Think like this:

* HTML → gives structure (building)
* CSS → paints the building
* JavaScript → makes everything move and work inside it

---

## 💡 6. Why JavaScript is important today?

JavaScript is powerful because:

* It runs in all browsers
* It is beginner-friendly
* It works everywhere (frontend + backend)
* It is one of the most used languages in the world

---

// Non Primitive Data Types

person = {
  name: "Abishek",
  age: 23,
  isStudent: true,
};

console.log(typeof person);

console.log(person);

console.log(person.name);
console.log(person.age);
console.log(person.isStudent);

person.name = "Sathiyan";
person.age = 50;
person.isStudent = false;

console.log(person.name);
console.log(person.age);
console.log(person.isStudent);

let numbers = [1, 2, 3, 4, 5];
console.log(typeof numbers);

numbers.pop();
console.log(numbers);

numbers.shift();
console.log(numbers);

let name = ["Abishek", "Sathiyan"];

console.log(name);
