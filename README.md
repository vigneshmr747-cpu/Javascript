## 📅 Day 1 – Basics

### 🧠 What We Learned

#### 📌 Variables

Variables are used to **store data**.

* `let` → value can change
* `const` → value cannot change
* `var` → old method (not recommended)

---

#### 🖨️ Printing Output

We use `console.log()` to print output.

```javascript
console.log("Hello World");
```

---

#### 📦 Datatypes

### ✅ Primitive Datatypes

* **String** → `"Hello"`
* **Number** → `10`
* **Boolean** → `true / false`
* **Null** → empty value
* **Undefined** → declared but no value

### ✅ Non-Primitive Datatypes

* **Array** → `[1, 2, 3]`
* **Object** → `{ name: "Abi", age: 22 }`

---

## 💻 Code Example

```javascript
// Variables
let name = "Abishek";
const age = 22;
var city = "Chennai";

// Printing
console.log("Hello World");
console.log(name);
console.log(age);

// Datatypes
let str = "Hello";
let num = 10;
let bool = true;
let x = null;
let y;

// Array
let arr = [1, 2, 3];

// Object
let obj = {
  name: "Abi",
  age: 22
};

// Output
console.log(str, num, bool, x, y);
console.log(arr);
console.log(obj);
```
